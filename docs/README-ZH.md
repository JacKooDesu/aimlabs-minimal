# AimLabs 極簡版

我是欠缺練習的 FPS 遊戲愛好者，之前用 AimLabs 來提升技術。但它現在又大又圓，廣告多而且大小來到了 10 多 GB。

所以這就是為什麼有了 AimLabs 極簡版，簡稱 ALM！！

## 功能特色

- 檔案體積極小且非營利的訓練工具
- 可透過 js 腳本擴充任務
- 計劃開發 WebGL 版本，意味著你不需要下載
